林林总总¶

为什么做这个站点？¶

时至今日，宗教仍然是这个世界上许多人的精神家园，他们笃信神和教义，并从中获得依靠和力量。

而很多如我的无神论者，该如何同样去寻找精神的寄托？ 也许我们可以为无神论者也创建一种信仰，这种信仰的神就是个人的自由意志（虽然人是否有自由意志是存疑的，但这里将其作为基石），其信条就是各种原则。 我认为这种信仰介于正经的宗教和飞天面条教之间。

我个人没有信仰任何宗教的经验，所以对于宗教的建构并不熟悉。希望某个机缘巧合的机会，有某位了解宗教的朋友看到这个站点，也许可以从中获得部分灵感。

从很久之前我就开始记录一些人生经验，起先是放在Google Keep，但是因为墙的原因并不能稳定访问，所以也并不能保持经常阅读。 去年我通过喜马拉雅听了《原则》，把这些零碎的经验重新做了整理。起先是放在Onedrive里的一个Excel文件，不过展示效果也并不好。 直到我发现了Sphinx-doc、 github和ReadTheDocs三件套。

  • Sphinx-doc可以很方便的生成整洁美观的文档

  • github可以很方便的进行版本控制以及分叉，便于在将来向不同方向的发展和调整

  • ReadTheDocs能够自动托管文档

怎么来做这个站点？¶

  1. 安装Python

  2. 安装Sphinx-doc和sphinx-rtd-theme

pip install Sphinx

pip install sphinx-rtd-theme

  1. 安装Sublime Text

  2. 安装Git

  3. 注册Github和ReadTheDocs

  4. 书写文档并推送到Github和ReadTheDocs
